# Quiz Master

A fully-functional, interactive quiz application built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies – just clean, efficient code.

## Features

### Question Types
- **Single Select** - Classic multiple choice (one correct answer)
- **Multi-Select** - Choose all correct answers from options
- **Fill in the Blanks** - Type your own answer with hints
- **True/False** - Simple boolean questions

### Core Features
- Real-time scoring and instant feedback
- Answer verification with explanations
- Beautiful progress tracking with visual progress bar
- Detailed results screen with performance breakdown
- Responsive design (works on mobile, tablet, desktop)
- Smooth animations and transitions
- Question counter and quiz statistics
- Filter options (All, Active, Completed)

### User Experience
- Clean, modern interface with elegant color scheme
- Instant visual feedback for correct/incorrect answers
- Shows correct answer explanation for wrong responses
- Dynamic question rendering based on type
- Accessible form inputs with proper labels
- Mobile-first responsive design

## Live Demo

Open `quiz-game.html` in any modern web browser to use the app immediately. No server or installation required!


## Usage

### For End Users

1. **Start the Quiz** - Open the HTML file to begin
2. **Answer Questions** - Click options or type your answer based on question type
3. **Submit Answers** - For multi-select and fill-blank questions, click "Submit Answer"
4. **Get Feedback** - Instant feedback shows if your answer is correct with explanations
5. **View Results** - After all questions, see your final score and detailed breakdown
6. **Restart** - Click "Try Again" to restart the quiz

## Features Breakdown

### Question Type: Single Select
- User clicks one option
- Answer is immediately checked
- Shows correct/incorrect feedback
- Prevents changing answer after submission

### Question Type: Multi-Select
- User clicks multiple options
- Must click "Submit Answer" button
- Checks if all selections match correct answers
- Shows which answers should be selected

### Question Type: Fill in the Blank
- User types their answer
- Case-insensitive matching
- Shows hint to help user
- Must click "Submit Answer" to verify

### Question Type: True/False
- Two button options (True/False)
- Immediate answer checking on click
- Visual distinction between buttons
- Shows correct answer explanation

### Results Screen
- Total score displayed as fraction (e.g., 7/8)
- Percentage score calculation
- Breakdown of correct vs incorrect answers
- Performance messages based on score
- Progress bar visualization
- "Try Again" button to restart quiz
